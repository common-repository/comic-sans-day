Do you want to join the official Comic Sans Day? Now you can, activate this plug-in and lean back on Comic Sans Day!

== Description ==

Every first friday of July it's the official Comic Sans Day.

With this plug-in your Wordpress website will have automatically Comic Sans on  Comic Sans Day   

== Installation ==

This section describes how to install the plugin and get it working.

* Download the plugin (or search for Comic Sans Day in the WordPress plugin database)
* Upload 'comic-sans-day' to the '/wp-content/plugins/ directory
* Activate the plugin through the 'Plugins' menu in WordPress
* Enjoy Comic Sans Day

== Frequently Asked Questions ==

= When will it be activated =

On the First friday of July

= Does this work every year =

Yes, you don't need to update. It works every year!

= Is there a preview option =

Yes, when the plug-in is activated you will see next to Deactivate a link with Preview. Just click it.
This will only be visible by you when logged in as admin
